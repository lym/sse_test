# Server Sent Events Example

Created this to test the state of HTTP streaming in Rails 4
Based on @tenderlove awesome blog intro

## Ruby version
  Ruby 2.0.0-p0

## System dependencies
  Linux, since I used rb-inotify which is a wrapper around inotify, which is a linux utility.

## Configuration
  <li>Get the code </li>
  <li>Bundle</li>
  <li>Fire up the server, Puma </li>
  <li>Curl -i "url" </li>
  <li>Modify a file in your set directory and check the results out in the shell window that's running curl </li>
